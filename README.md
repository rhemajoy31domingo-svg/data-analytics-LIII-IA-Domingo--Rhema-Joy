## Project Overview: Bahaghari Retail Co. Data Scenario
Bahaghari Retail Co. is a growing retail enterprise navigating a highly competitive market, requiring a transition from intuition-driven operations to robust, data-backed decision-making. This project focuses on auditing, cleaning, and transforming a disorganized multi-table transactional dataset—consisting of raw transaction records, product details, and store profiles—plagued with duplicated rows, inconsistent text strings, missing regional tags, and unformatted data types. By executing comprehensive descriptive statistical modeling, database preparation, and dynamic visual layouts, this project uncovers hidden revenue drivers, tracks regional performance concentrations, evaluates product category margins, and provides leadership with actionable strategic recommendations to optimize their supply chain and regional footprint.

## Directory Structure
The repository is structured into the following designated folders to isolate each phase of the data lifecycle:

* **`01_statistics/`** Contains `IA_Statistics_Domingo_Rhema_Joy.xlsx`. This workbook utilizes the Excel Data Analysis ToolPak to calculate measures of central tendency (mean, median, mode) and dispersion (variance, standard deviation, range, quartiles), and includes a structured data histogram alongside a formal interpretation paragraph.
* **`02_data_preparation/`** Contains `IA_Preparation folder for SQL queries` and `IA_PreparationWorkbook_Domingo_Rhema_Joy.xlsx`. These files house the structural cleaning pipeline used to remove duplicate entries, correct structural dates, trim trailing spaces, resolve NULL values, and perform relational joins to establish a clean, unified dataset.
* **`03_visualization/`** Contains `IA_Dashboard_Domingo_Rhema_Joy.pbix` and `IA_PublishedLink_Domingo_Rhema_Joy.png`. This directory holds the active Power BI data model, DAX measures, structural star schema relationships, and interactive reporting dashboards.
* **`04_executive_summary/`** Contains `IA_Executive Summary_Domingo_Rhema_Joy.pdf`. A concise, one-page business-ready document detailing core statistical discoveries, major organizational operational findings, and high-impact strategic business recommendations for executive leadership.
